🎉 GitLog Widget Test Page

This page demonstrates how to embed the GitLog widget on your website

📋 Widget Installation Instructions

Copy the script below and paste it into your website's HTML, right before the closing </body> tag.

1. Embed Script

<script src="http://localhost:3000/widget.js" data-widget-id="YOUR_WIDGET_ID" async></script>

Replace YOUR_WIDGET_ID with the widget ID generated from your GitLog dashboard.

2. Widget Features

🎨 Customizable

Match your brand colors, position, and size

⚡ Fast Loading

Async loading won't slow down your site

📱 Mobile Responsive

Works perfectly on all devices

📊 Analytics Built-in

Track impressions and clicks automatically

3. Configuration Options

<script src="http://localhost:3000/widget.js" data-widget-id="YOUR_WIDGET_ID" data-position="bottom-right" data-size="medium" data-api-url="http://localhost:3000/api/widget" async ></script>

Available options:

  • data-position: bottom-right, bottom-left, top-right, top-left
  • data-size: small, medium, large
  • data-api-url: Your GitLog API URL (optional)

4. Testing the Widget

  1. Go to your GitLog dashboard at http://localhost:3000/widget
  2. Click "Generate Widget ID"
  3. Copy the script snippet
  4. Replace YOUR_WIDGET_ID in this page's source code
  5. Refresh the page to see the widget!
